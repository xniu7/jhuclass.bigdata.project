Feature

Gene Count

DNA has only four nucleobases called adenine(A), cytosine(C), guanine(G), thymine(T). We count each nucleobase by iterating each segment. By count the respective ACGT from the fastA file, we have an overview of composition of the genome.

Segment Length

DNA genome is huge, and we would always like to seperate it into segments for analysis. In fastA format file, all segments represents by the symbol '>' at the begining. With variety length, we count each segment and have a look at the distribution.

GC Content

The GC-content of a DNA string is given by the percentage of symbols in the string contains C of G. We compute GC-content of all segments and see the distributions.

Protein Unit

Either strand of a DNA double helix can serve as the coding strand for RNA transcription. Hence, a given DNA string mplies six total reading frames. An open reading frame (ORF) is one which starts fromt he start codon and ends by stop codon without any other stop codons in between. We compute the number of ORFs of all segments to show how many possible proteins can be translated.

Genome Assembly

In bioinformatics, sequence assembly refers to aligning and merging gragments of a much longer DNA sequence in orderreconstruct the original sequence. The is needed as DNA sequencing technology cannot read whole genomes in one go, but rather reads small pieces of between 20 and 1000 bases. Typically the short fragments called reads, result from shotgun sequencing genomic DNA, or gene transcript.

Ham / Edit Distance

Given two DNA string of equal length, the hamming distance is the number of corresponding symbols that differ in the two strings, which runs fast and tells the mistakes of these two DNA. On the other hand, the edit distance is the minimum number of edit operations needed to transform one string into another, where and edit operation is defined as the substitution, insertion, or deletion of a single symbol. We compute both Ham and Edit distance for two fastQ segments.
